Participating

The mobile phone application will be made available to anyone who would like to participate in skating routes. All you need is a smart-phone that supports Java (MIDP2) with the Bluetooth API (JSR-82). See a list of phones on www.j2mepolish.org. For GPS you can use a standard GPS bluetooth module. You can find a list of GPS BT modules on www.gpspassion.com.

Registration through this site is under development. Send an email to just AT justobjects.nl if you are interested.

You can also help by sending a map of tracks yo would like to see mapped. Send an image or document like this example (thanks Jan Pauwels!) to just AT justobjects.nl.
