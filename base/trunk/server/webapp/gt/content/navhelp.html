How to Navigate the Map

What you see is projected on a Google Map.

Zoom In/Out

Use the + and - icons in the upper left or the slider. Note that not all maps may support very detailed zoom levels at all places in the world. Double clicking anywhere on the map also zooms and centers the map.

Moving (Panning)

You can move (pan) the map by using the arrow icons in the upper left, but far easier is to drag the map with your mouse.

Centering

Double-clicking anywhere on the map centers and zooms the map at that point.

Selecting Maps

Choose a different map as background by using the options under the "Map" menu. Some map-types may contain specific data/features like GPS-trace points.

Advanced: passing page parameters

Initial map parameters for zoom, map center and map name, may be passed as page parameters accessing the main page.

For example: www.geotracing.com/gt?center=-2.25,53.4833&map=satellite&zoom=12
Allowed parameters are:

  • center: lon,lat value in fractional decimal
  • map: map name (see menu mapnames), e.g. "streets", "blanc"
  • zoom: Google Map2 zoomlevel (values 1-18, zoomed out to zoomed in)
