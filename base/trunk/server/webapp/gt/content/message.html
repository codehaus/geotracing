New Version MobiTracer

As of march 26, 2007 there is a new version (3.6.0) of the MobiTracer mobile client. If you have an older version, that may still work for making tracks but media uploads (photos/text/audio) will fail. Please login and install the latest version using the instructions provided under [mobitracer] once logged in.
